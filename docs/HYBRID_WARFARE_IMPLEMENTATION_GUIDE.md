## 1. تحديثات قاعدة البيانات

### جدول `so_news_events` - الحقول الجديدة المضافة (60+ حقل)

#### أ. حقول التصنيف المتقدم:
- `osint_type`: نوع OSINT (عسكري، أمني، سيبراني، سياسي، اقتصادي، إلخ)
- `hybrid_layers`: طبقات الحرب المركبة النشطة (JSON)
- `event_category`: تصنيف الحدث الرئيسي
- `strategic_category`: التصنيف الاستراتيجي
- `operational_level`: المستوى العملياتي

#### ب. حقول التأثير والوزن:
- `political_weight`: الوزن السياسي (0-100)
- `economic_weight`: الوزن الاقتصادي (0-100)
- `social_impact`: الأثر الاجتماعي (0-100)
- `cyber_impact`: الأثر السيبراني (0-100)

#### ج. حقول الفاعل وشبكة العلاقات:
- `primary_actor`: الفاعل الرئيسي
- `secondary_actor`: الفاعل الثانوي
- `actor_network`: شبكة الفاعلين (JSON)
- `actor_relationships`: علاقات الفاعلين (JSON)
- `sponsor_entity`: الجهة الراعية
- `funding_entity`: الجهة الممولة
- `media_operator`: المشغل الإعلامي

#### د. الحقول الجغرافية المتقدمة:
- `geo_country`, `geo_region`, `geo_city`, `geo_district`
- `geo_coordinates`: الإحداثيات (lat,lon)
- `geo_accuracy`: دقة الموقع
- `geo_sensitivity`: حساسية الموقع

#### هـ. حقول الزمن الدقيق:
- `event_start_time`, `event_end_time`
- `publish_delay`: فارق الزمن بين الحدث والنشر
- `time_accuracy`: دقة التوقيت

#### و. حقول التحقق المتقدم:
- `verification_status`: حالة التحقق (غير مؤكد، محتمل، مؤكد)
- `verified_sources_count`: عدد المصادر المؤكدة
- `has_visual_evidence`: وجود أدلة بصرية
- `has_satellite_imagery`: وجود صور أقمار صناعية
- `has_official_statement`: وجود بيان رسمي
- `source_conflict`: وجود تضارب في المصادر
- `verification_notes`: ملاحظات التحقق

#### ز. حقول Scores المتقدمة:
- `sentiment_score`: درجة التحليل العاطفي (-100 إلى +100)
- `threat_score`: درجة التهديد (0-100)
- `escalation_score`: درجة التصعيد (0-100)
- `confidence_score`: درجة الثقة (0-100)
- `stability_index`: مؤشر الاستقرار
- `aggression_index`: مؤشر العدوانية
- `risk_level`: مستوى الخطر (منخفض، متوسط، مرتفع، حرج)
- `impact_radius`: نطاق التأثير (محلي، إقليمي، دولي)
- `urgency_level`: مستوى الاستعجال

#### ح. حقول النية والسياق:
- `probable_intent`: النية المرجحة
- `probable_goal`: الهدف المرجح
- `political_driver`, `military_driver`, `economic_driver`, `media_driver`
- `trigger_event`: الحدث المحفز
- `general_context`: السياق العام
- `linked_previous_event`: الرابط مع حدث سابق

#### ط. حقول النمط والتحليل:
- `pattern_type`: نوع النمط
- `pattern_frequency`: تكرار النمط
- `trend_direction`: اتجاه الاتجاه
- `cycle_detected`: اكتشاف دورة متكررة
- `signature_behavior`: سلوك بصمة الفاعل
- `escalation_chain_id`: معرف سلسلة التصعيد

#### ي. حقول التوقع والإنذار:
- `likely_scenario`: السيناريو المرجح
- `alternative_scenario`: السيناريو البديل
- `prediction_timeframe`: الإطار الزمني للتوقع
- `prediction_confidence`: درجة ثقة التوقع
- `escalation_probability`: احتمالية التصعيد
- `containment_probability`: احتمالية الاحتواء
- `spread_probability`: احتمالية الانتشار الجغرافي

#### ك. حقول الإنذار:
- `alert_flag`: علم الإنذار
- `alert_type`: نوع الإنذار
- `alert_reason`: سبب الإنذار
- `alert_threshold`: عتبة الإنذار
- `alert_priority`: أولوية الإنذار
- `alert_status`: حالة الإنذار

#### ل. حقول الحرب المركبة الإضافية:
- `warfare_layers`: طبقات الحرب المركبة النشطة
- `multi_domain_score`: درجة التعددية المجال
- `strategic_impact`: الأثر الاستراتيجي
- `asymmetric_indicator`: مؤشر حرب غير متكافئة
- `cognitive_warfare_flag`: علم الحرب المعرفية
- `information_operation`: عملية معلوماتية

### مفاتيح الفهرسة الجديدة:
- `idx_verification`: للبحث حسب حالة التحقق ودرجة الثقة
- `idx_threat`: للبحث حسب درجة التهديد وعلم الإنذار
- `idx_hybrid`: للبحث حسب التعددية المجال ومستوى الخطر

## 2. قاموس طبقات الحرب المركبة (9 طبقات)

تم تعريف قاموس كامل لطبقات الحرب المركبة مع الكلمات المفتاحية لكل طبقة:

1. **الطبقة العسكرية**: غارات، قصف، استهداف، هجمات، اشتباكات
2. **الطبقة الأمنية**: اعتقالات، مداهمات، تفكيك خلايا
3. **الطبقة السيبرانية**: اختراقات، تسريبات، هجمات رقمية
4. **الطبقة الجغرافية**: صور أقمار صناعية، تحصينات، تدمير
5. **الطبقة السياسية**: تصريحات، عقوبات، مفاوضات
6. **الطبقة الاقتصادية**: نفط، غاز، موانئ، أسواق
7. **الطبقة الاجتماعية**: احتجاجات، تحريض، حرب نفسية
8. **طبقة الطاقة**: منشآت نفطية، كهرباء، مرافق حيوية
9. **طبقة الممرات الاستراتيجية**: مضائق، موانئ، قواعد

## 3. المحركات التحليلية الجديدة

### أ. محرك تصنيف طبقات الحرب المركبة
```php
sod_classify_hybrid_layers($title, $content)
```
- يحلل النص لتحديد الطبقات النشطة
- يحسب درجة كل طبقة
- يحدد إذا كان الحدث متعدد المجالات

### ب. محرك حساب Scores المتقدم
```php
sod_calculate_advanced_scores($event_data)
```
- يحسب Sentiment Score بناءً على اللغة المستخدمة
- يحسب Threat Score بناءً على مؤشرات التهديد
- يحسب Escalation Score بناءً على كلمات التصعيد
- يحسب Confidence Score بناءً على مصادر التحقق
- يحدد Risk Level و Urgency Level

### ج. محرك استخراج شبكة الفاعلين
```php
sod_extract_actor_network($title, $content)
```
- يستخرج الفاعل المنفذ
- يحدد الجهات الراعية والممولة
- يكشف المشغلين الإعلاميين
- يبني شبكة العلاقات بين الكيانات

### د. محرك التحقق المتقدم
```php
sod_advanced_verification($event_data)
```
- يفحص مصادر موثوقة (رويترز، AP، FP، إلخ)
- يكشف الأدلة البصرية والفضائية
- يتحقق من البيانات الرسمية
- يرصد التضارب في الروايات
- يحسب درجة ثقة شاملة

### هـ. محرك الدمج التحليلي
```php
sod_enhance_event_with_hybrid_analysis($event_data)
```
- يدمج جميع التحليلات في حدث واحد
- ينتج بيانات جاهزة للتخزين والعرض

## 5. طريقة التفعيل

### التفعيل التلقائي:
عند تنشيط الإضافة أو تحديثها، سيتم تنفيذ الدالة التالية تلقائياً:
```php
sod_activate_hybrid_warfare_update()
```

هذه الدالة تقوم بـ:
1. إضافة جميع الحقول الجديدة لقاعدة البيانات
2. إنشاء مفاتيح الفهرسة
3. تسجيل إصدار التحديث
4. كتابة سجلات التفعيل

### التفعيل اليدوي (إذا لزم الأمر):
```php
// في functions.php أو وحدة تحكم مخصصة
require_once 'path/to/osint-hybrid-warfare-update.php';
sod_hybrid_warfare_add_advanced_fields();
```

## 7. أمثلة على الاستخدام

### مثال 1: تحليل حدث إخباري
```php
$event = [
    'title' => 'غارة إسرائيلية تستهدف منشأة نفطية في سوريا',
    'war_data' => 'أعلنت مصادر محلية عن غارة جوية...',
    'actor_v2' => 'جيش العدو الإسرائيلي',
    'region' => 'سوريا',
    'score' => 150
];

$enhanced = sod_enhance_event_with_hybrid_analysis($event);
// النتيجة تحتوي على جميع الحقول المحسنة
```

### مثال 2: الاستعلام عن أحداث عالية الخطورة
```sql
SELECT * FROM wp_so_news_events 
WHERE threat_score >= 60 
AND alert_flag = 1 
AND event_timestamp >= UNIX_TIMESTAMP(NOW() - INTERVAL 24 HOUR)
ORDER BY threat_score DESC;
```

### مثال 3: تصفية حسب طبقات الحرب المركبة
```php
// الحصول على الأحداث متعددة المجالات
$multi_domain_events = $wpdb->get_results(
    "SELECT * FROM {$table} 
     WHERE multi_domain_score >= 30 
     AND hybrid_layers IS NOT NULL"
);
```

## 9. الخطوات التالية الموصى بها

1. **اختبار التحديثات**: تشغيل الإضافة في بيئة اختبار والتحقق من عمل جميع الحقول

2. **تطوير الواجهات**: إنشاء شاشات عرض متخصصة للاستفادة من الحقول الجديدة

3. **التكامل مع APIs خارجية**: ربط النظام بمصادر بيانات إضافية (صور أقمار صناعية، RSS feeds)

4. **تدريب الفريق**: شرح كيفية استخدام المؤشرات الجديدة في التحليل اليومي

5. **إنشاء تقارير مخصصة**: تطوير قوالب تقارير تستفيد من بيانات الحرب المركبة
