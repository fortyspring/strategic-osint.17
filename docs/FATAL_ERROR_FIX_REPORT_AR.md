## السبب الجذري
كان ملف `class-security-loader.php` يستدعي دوالاً وإجراءات (Actions) مباشرة عند تحميل الملف، قبل أن يتم تهيئة WordPress بشكل كامل. هذا تسبب في:
1. استدعاء `osint_initialize_security()` تلقائياً عند التحميل
2. تسجيل إجراءات `add_action()` قبل الأوان
3. مشاكل في ترتيب تحميل الملفات
