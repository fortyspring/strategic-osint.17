# 🛡️ نظام حارس البوابة الذكي (Smart Gatekeeper)

## نظرة عامة
نظام متقدم لفلترة الأخبار الواردة لنظام Beiruttime OSINT، يهدف إلى منع "الضوضاء" الإخبارية (محتوى ترويجي، إعلانات، محتوى غير ذي صلة) والسماح فقط بـ "الإشارة" الاستخباراتية ذات القيمة.

---

## 📋 الميزات الرئيسية

### 1. فلترة ثلاثية الطبقات
- **المنع الصارم (Hard Blacklist)**: رفض فوري لأي خبر يحتوي على كلمات مثل (كازينو، وظائف، مسابقات، تخفيضات...).
- **فلتر الضوضاء (Noise Filter)**: حساب درجة الضوضاء بناءً على كلمات التسويق والإثارة الرخيصة.
- **فلتر الإشارة (Signal Filter)**: منح نقاط للكلمات الدلالية العسكرية، الأمنية، السياسية، الاقتصادية حسب طبقات الحرب المركبة.

### 2. قواميس موسعة وجاهزة
- **قاموس المنع**: 50+ كلمة ممنوعة.
- **قاموس الضوضاء**: 15+ كلمة تشير لمحتوى ضعيف.
- **قاموس الإشارة**: 100+ كلمة دلالية موزعة على 7 طبقات (عسكرية، أمنية، سيبرانية، سياسية، اقتصادية، جغرافية، اجتماعية).

### 3. نظام نقاط ذكي
- كل كلمة إشارة لها وزن مختلف (مثلاً: "قتيل" = 5 نقاط، "اجتماع" = 1 نقطة).
- الخبر يُقبل فقط إذا تجاوز عتبة الإشارة الدنيا (افتراضياً 15).
- الخبر يُرفض فوراً إذا تجاوز عتبة الضوضاء القصوى (افتراضياً 40).

### 4. تطبيع النصوص المتقدم
- إزالة التشكيل العربي.
- توحيد الألف والياء والتاء المربوطة.
- تجاهل الروابط URLs.
- دعم اللغتين العربية والإنجليزية.

### 5. سجلات وتتبع
- تسجيل جميع الأخبار المقبولة والمرفوضة في `error_log`.
- حفظ آخر 100 خبر مرفوض في قاعدة البيانات (Transient) للمراجعة.
- واجهة في لوحة التحكم لعرض المرفوضات وأسباب الرفض.

### 6. إدارة ديناميكية من لوحة التحكم
- تعديل عتبات القبول والرفض.
- إضافة كلمات مخصصة للقائمة السوداء.
- إضافة إشارات مخصصة جديدة.
- أداة اختبار يدوي لفحص أي خبر قبل اعتماده.

## 🚀 طريقة التثبيت والاستخدام

### 1. تضمين الملف
أضف السطر التالي في الملف الرئيسي للإضافة (`beiruttime-osint-pro.php`):

```php
require_once __DIR__ . '/class-smart-gatekeeper.php';
```

### 2. التكامل مع عملية جمع الأخبار
في الدالة المسؤولة عن معالجة الأخبار الجديدة (قبل الحفظ في قاعدة البيانات)، استخدم الدالة المساعدة:

```php
$result = sod_intercept_news_ingestion($title, $content, $source);

if (is_wp_error($result)) {
    // الخبر مرفوض
    error_log('Filter Blocked: ' . $result->get_error_message());
    return false; // إيقاف الحفظ
}

// الخبر مقبول، أكمل الحفظ
$event_data = [
    'title' => $result['title'],
    'content' => $result['content'],
    'gatekeeper_score' => $result['gatekeeper_score'],
    // ... بقية الحقول
];
// save_to_database($event_data);
```

### 3. الوصول للوحات التحكم
بعد التفعيل، ستجد قائمتين جديدتين تحت قائمة `Beiruttime OSINT`:
1. **سجل المرفوضات**: لعرض الأخبار التي تم منعها وسبب المنع.
2. **إعدادات الحارس**: لضبط العتبات وإدارة القواميس واختبار الفلتر.

## 📊 هيكل القواميس

### أ. القائمة السوداء الصارمة (Hard Blacklist)
تشمل: القمار، الوظائف، المسابقات، الإعلانات التجارية، العقارات، الطبخ، الأبراج، النكت، الرياضة البحتة.

### ب. قاموس الضوضاء (Noise Keywords)
يشمل: عبارات clickbait مثل "شاهد بالفيديو"، "صدمة"، "انتشار واسع"، "اضغط هنا".

### ج. قاموس الإشارة (Signal Dictionary)
موزع حسب طبقات الحرب المركبة:
- **عسكرية**: غارة، صاروخ، اشتباك، قتيل...
- **أمنية**: اعتقال، تفكيك خلية، تجسس...
- **سيبرانية**: اختراق، تسريب بيانات، هجوم رقمي...
- **سياسية**: عقوبات، قمة، مفاوضات، بيان...
- **اقتصادية**: نفط، غاز، كهرباء، ميناء، عملة...
- **جغرافية**: قمر صناعي، إحداثيات، مضيق، حدود...
- **اجتماعية**: احتجاج، نزوح، تحريض، دعاية نفسية...
- **فاعلين**: أسماء دول، جيوش، فصائل، مدن حساسة.

## ⚙️ الإعدادات الافتراضية

| الإعداد | القيمة | الوصف |
|---------|--------|-------|
| الحد الأدنى للإشارة | 15 | يجب أن يحصل الخبر على 15 نقطة على الأقل |
| الحد الأقصى للضوضاء | 40 | إذا تجاوزت الضوضاء 40 يُرفض فوراً |
| وزن كلمة "قتيل" | 5 | قيمة عالية لأهمية الحدث |
| وزن كلمة "اجتماع" | 1 | قيمة عادية |

يمكن تغيير هذه القيم من صفحة **إعدادات الحارس**.

## 🧪 أمثلة على الفلترة

### مثال 1: خبر مرفوض (إعلاني)
**العنوان**: "مسابقة كبرى! اربح رحلة مجانية تابعونا الآن"
**النتيجة**: ❌ مرفوض فوراً (Hard Blacklist: مسابقة، اربح، تابعونا).

### مثال 2: خبر مرفوض (ضعف المحتوى)
**العنوان**: "شاهد بالفيديو: منظر غريب في السماء ينتشر بشكل واسع"
**النتيجة**: ❌ مرفوض (ضوضاء عالية جداً، لا توجد إشارات كافية).

### مثال 3: خبر مقبول (عسكري)
**العنوان**: "غارة جوية تستهدف مستودع ذخائر شمال سوريا وقتيل وجريح"
**النتيجة**: ✅ مقبول (درجة إشارة عالية: غارة=5، مستودع=1، قتيل=5، جريح=4 = 15 نقطة).

### مثال 4: خبر مقبول (سياسي/اقتصادي)
**العنوان**: "عقوبات اقتصادية جديدة على قطاع النفط الإيراني تؤثر على أسعار الطاقة"
**النتيجة**: ✅ مقبول (عقوبات=4، نفط=3، طاقة=3، إيراني=1 = 11 نقطة + سياق).
*ملاحظة: يمكن خفض العتبة أو إضافة كلمات سياقية لرفع الدرجة.*
